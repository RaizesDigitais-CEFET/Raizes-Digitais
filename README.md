# 🌱 Raízes Digitais — PJ050-2026

> **Cultivating Innovation with Generative AI in South-Minas Coffee Growing**

Extension project by **CEFET-MG** (Varginha campus) in partnership with **Minasul cooperative**.

---

## 📌 About the Project

Raízes Digitais deploys and validates a generative AI assistant — based on **LLMs + RAG** — to support coffee farmers and technicians in the South Minas region of Brazil.

**Two main goals:**
1. 🤖 Deliver a functional agricultural chatbot (PoC) — by **2026-08-31**
2. 🎓 Certify **50 producers/technicians** in a practical AI literacy workshop — by **2026-11-30**

**Period:** May 2026 – December 2026  
**Status:** 🟢 In execution

---

## 🏗️ Tech Stack

| Layer | Technology |
|---|---|
| Orchestration / Automation | n8n, Neurônios |
| LLMs | Gemini (+ other APIs) |
| Persistence / Memory | Supabase (PostgreSQL) |
| Knowledge Base | Notion |
| User Interface | Telegram Bot |
